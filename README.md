# 🗑️ Rubbish Rush

A fast-paced, mobile-friendly single-page web game that teaches New Zealanders how to correctly sort rubbish into the right bins, based on NZ's 2023 national kerbside collection standards.

## 🎮 Play Now

**[▶ Play Rubbish Rush on GitHub Pages](https://ittytheitstar.github.io/RubbishRush/)**

## How to Play

- **Drag** (or **tap**) each rubbish item into the correct bin before the clock runs out.
- Use the **action buttons** (Rinse, Remove Lid, Flatten…) before sorting to earn bonus time.
- Forgetting a required action (e.g. removing a bottle lid) costs you time!
- Build **combos** by sorting correctly in a row for mega bonuses.
- The timer adds pressure — the closer you get to zero, the more intense it gets!

## NZ Bin Rules

| Bin | What goes in |
|-----|-------------|
| 🌿 **Green** | Food scraps, garden waste, compostable material |
| ♻️ **Yellow** | Clean, dry rigid plastics (1, 2, 5), glass bottles & jars, steel/aluminium cans |
| 🗑️ **Red** | Non-recyclables, soft plastics, polystyrene, contaminated items |
| 📄 **Blue** | Newspapers, cardboard (flattened), paper, magazines |

## Features

- 🇳🇿 43 NZ-specific items (Milo tin, L&P bottle, Weetbix box…)
- ⏱️ Countdown timer with urgency effects (pulse, glow, shake)
- 🔊 Sound effects via Web Audio API
- 🏆 High score saved in local storage
- 📱 Mobile-friendly drag + tap-to-sort controls
- ✨ Combo multipliers and animated feedback

## Tech

Pure HTML + CSS + JavaScript — no build step required. Hosted on GitHub Pages.
